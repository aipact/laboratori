# Gruppo 2 - Gestione Automatizzata Segnalazioni Cittadini

## Tema
Sistema di ingestion multicanale per segnalazioni cittadine con smistamento AI e sintesi risposte.

## Obiettivo
Automatizzare la gestione delle segnalazioni provenienti da:
- Email
- WhatsApp
- Portale Web

Con smistamento intelligente agli uffici competenti ed elaborazione sintesi risposte.

## Contesto
Comune capoluogo di provincia - Toscana

## Output Prodotto
- `gruppo2.html`: Prototipo interfaccia operatore con mockup interattivo

## Caratteristiche Tecniche
- HTML5 con Mermaid.js per diagrammi di flusso
- Mockup interattivo con scenari (normale/conflitto)
- Dashboard split-screen per validazione
- Sistema di alert per contraddizioni

## Funzionalita Chiave
1. **Normalizzatore AI**: Converte input multicanale in ticket unico
2. **AI Classifier**: Analisi semantica per smistamento
3. **Matrice Competenze**: Routing automatico agli uffici
4. **AI Composer**: Generazione bozza risposta con check logico
5. **Human-in-the-loop**: Validazione finale operatore

## Requisiti Funzionali Implementati
- FR-002: Entity Extraction (Chi, Dove, Cosa)
- FR-012: Logic Check Anti-Contraddizione
- FR-014: Dashboard Split-Screen

## Competenze Coinvolte
- CED/Sistemi Informativi
- Staff Comunicazione
- URP
